How to find the point of intersection!

1. First we need the equations of the two lines.
2. Then, because at the point of intersection the two equations will have the same values of x and y, we set the two equations equal to each other. This gives an equation that we can solve for x.
3. Finally, we substitute that x value in one of the line equations (it doesn't matter which) and solve it for y.
